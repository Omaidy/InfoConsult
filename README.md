Proyecto Dynamic Web con JSP y Java
Requisitos previos
Apache Tomcat 9.x
MySQL 5.7+
JDK 8+
Instrucciones
Importar la base de datos:

Importa el archivo dwi.sql en tu servidor MySQL.
Configuración de la base de datos:

La conexión predeterminada está configurada para el puerto 3306, usuario root sin contraseña.
Si tu configuración es diferente, actualiza estos parámetros en el archivo de configuración del proyecto.
Iniciar el proyecto:

Despliega el proyecto en Apache Tomcat.
Accede a la aplicación en: http://localhost:8080/InfoConsult/SvParrillaMeses?mes=&ano=
